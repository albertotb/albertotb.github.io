--- title: "Test if port 22 is open" date: '2013-10-01T19:18:00.002+02:00' tags: modified_time: '2013-10-01T19:20:40.921+02:00' blogger_id: tag:blogger.com,1999:blog-5062930559472516636.post-2065820211862960577 blogger_orig_url: http://sk-forgetmenot.blogspot.com/2013/10/bash-test-if-port-22-is-open.html --- With this script you can check if a remote machine is listening on port 22, which is the default SSH port. {% highlight bash %} port=22 timeout=5 if (( "$#" != 1 )); then echo "usage: $(basename $0) HOST" fi host=$1 if nc -w $timeout -z $host $port; then echo "Yes" else echo "No" fi {% endhighlight %}
