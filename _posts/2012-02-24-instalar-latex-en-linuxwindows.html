--- title: Instalar Latex en Linux/Windows date: '2012-02-24T15:14:00.002+01:00' tags: - latex texmaker ubuntu texlive modified_time: '2012-06-28T12:35:28.787+02:00' blogger_id: tag:blogger.com,1999:blog-5062930559472516636.post-2500887431677799888 blogger_orig_url: http://sk-forgetmenot.blogspot.com/2012/02/instalar-latex-en-linuxwindows.html --- Para escribir en LaTeX en general se necesitan dos cosas, la distribución de LaTeX y un editor.

Linux (Debian/Ubuntu)

La distribución más común es texlive. En Debian/Ubuntu se puede instalar con:

sudo apt-get install texlive

Una vez instalada, se puede escribir el fichero .tex con cualquier editor y compilar (en pdf) con pdflatex. Sin embargo, en general es más cómodo instalar un editor específico para LaTeX (entorno de desarrollo). Un ejemplo es Texmaker, que se puede instalar con:

sudo apt-get install texmaker

Existen muchos bastante parecidos: Kile (para KDE), plugins para el vim, etc. La ventaja de Texmaker es que es multiplataforma, y por lo tanto se pueden instalar también en Windows.

Para instalar nuevos paquetes se pueden buscar en el repositorio oficial de LaTeX http://ctan.org/ y manualmente copiar los ficheros en los directorios adecuados (ver instrucciones detalladas). Sin embargo, existe un truco para no tener que instalar los nuevos paquetes manualmente, y es buscarlos en los repositorios oficiales. Para ello basta con poner:

sudo apt-cache search $nombre_paquete_latex

En la mayoría de las ocasiones aparecerá un paquete que contiene el paquete LaTeX buscado (entre otros). La desventaja de esto es que no se instala únicamente el paquete deseado y que muchas veces no es la última versión. La principal ventaja es que a partir de entonces estos paquetes se actualizarán automáticamente con apt.

Windows

En Windows la distribución de LaTeX más común es Miktex. Esta distribución viene con un editor propio, aunque no es demasiado intuitivo. Otra opción es TexMaker, que está disponible también para Windows. La ventaja de Miktex es que se puede configurar para que baje paquetes de LaTeX automáticamente, por lo que al compilar nunca faltará ninguno.
